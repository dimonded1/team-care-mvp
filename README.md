# Команда заботы — MVP

Отдельный greenfield-проект мобильного интерактива программы «Опека» фонда НИКА. Старый событийный прототип не используется как техническая или визуальная основа; перенесены только проверенные фотографии животных как контент.

## Сценарий

1. Загрузка.
2. Стартовый экран.
3. 11 свайп-вопросов: вправо/«Да», влево/«Нет».
4. Мэтч и паспорт одного животного.
5. Четыре ситуации: питание, здоровье, доверие, шанс на дом.
6. Объяснение опеки и команды до пяти опекунов.
7. Финал, CTA и PNG-карточка для сохранения/шеринга.

В MVP нет backend и Битрикс24. Локально сохраняется только анонимный прогресс.

## Локальный запуск

```bash
npm ci
npm run dev
```

## Проверки

```bash
npm run test
npm run build
```

После запуска Docker полный мобильный smoke-тест (реальный свайп, 11 вопросов, четыре ситуации и генерация карточки) выполняется так:

```bash
npm run smoke
```

Скриншоты проверки сохраняются локально в `artifacts/` и не коммитятся.

## Docker

```bash
docker compose build
docker compose up -d
```

Приложение будет доступно на `http://localhost:8080`. Runtime использует non-root nginx, read-only filesystem, healthcheck и security headers.
