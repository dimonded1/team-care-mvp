# Start screen override — Living launch console

This page intentionally diverges from the quieter passport screens. It is the
single high-energy entrance to the experience; every later screen keeps the
existing Living Editorial Passport system.

## Composition

- Full-viewport dark forest stage using the existing NIKA green.
- One centered launch console containing the complete value proposition and CTA.
- Four original vector cat/dog mascots orbit the console; no real animal photos
  appear before matching.
- Oversized background words, two orbital guide lines and a warm radial halo
  create depth without WebGL or raster effects.
- Mobile keeps the console fully legible while mascots peek from the corners.

## Interaction

- Primary CTA: “Найти своего подопечного”.
- Mascots use slow transform-only floating motion and blinking.
- Desktop pointer movement shifts mascots at different depths.
- Orbit lines move independently; all continuous movement stops under
  prefers-reduced-motion.
- The CTA remains a semantic button with visible keyboard focus and a 64–72px
  touch target.

## Visual rules

- Palette: #234721, #183419, #F76D31, #FFB5B5, #F6F2E9.
- Typography: bundled Raleway only.
- Central surface is warm and opaque enough for reliable contrast; no generic
  glassmorphic SaaS card.
- Vector mascots are decorative and hidden from assistive technology.
- Real photographs begin only after matching, when a specific animal matters.
